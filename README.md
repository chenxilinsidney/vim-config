Installation
---------------
   1. Clone the repo:
    * git clone git://github.com/chenxilinsidney/vim-config.git ~/.vim-config
   2. Install:
    * cd ~/.vim-config
    * cp -rf .vim ~
    * cp -f .vimrc ~
